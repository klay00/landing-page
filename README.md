## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/shantoislam6/nuxt-shadcn-landing.git
```

2. Go into project

```bash
cd nuxt-shadcn-landing
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
